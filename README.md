This is a RESTful API built using the MERN stack (MongoDB, Express, Node.js) for managing projects and tasks.

### Key Features:

* **Authentication (JWT):** Secure user login and registration.
* **Role-Based Access Control (RBAC):** Separate permissions for 'user' and 'manager' roles.
* **Project Management:** CRUD operations for projects, linked to the creator.
* **Task Management:** CRUD operations for tasks, linked to a project and assigned user.
